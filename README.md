# Taller-funciones-
trabajo de ejercicios para las tuplas diccionarios y funciones 

#Funciones detalladas
Funciones del taller
sum: Es una suma de todos los elementos de una lista 
len: Determina el número de elementos dentro de una lista o una lista 
max: Encuentra el valor mas grande de una lista o una tupla 
min: Encuentra el valor mas pequeño de una lista 
print: Muestra la información deseada en la consola 
input: Permite que el usuario escriba datos 
.get: usada en diccionarios busca una llave y si no existe regresa el valor 
.items: Se utiliza para obtener una pareja de datos 
.split: Corta un string y la convierte en una lista, por ejemplo se utilizó para separar las frases en el ejercicio 5 
def: sirve para crear e identificar funciones personalizadas
if: sirve para determinar si aglo es verdadero o falso 
 f”: Permite combinar variables directamente sin necesidad de especificar las sumas todo el tiempo
